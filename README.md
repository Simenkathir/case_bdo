# Case BDO

This is a .NET Core console application which retrieves latest data from the API: https://www.twelvedata.com/ at a regular interval or when receiving an HTTP request.

## Prerequisites

- .NET Core SDK
- A text editor or IDE like Visual Studio Code

## Setup

1. **Clone the repository**:
    git clone https://github.com/Simenkathir/case_bdo.git
    cd ConsoleApp

2. **Restore dependencies**:
    dotnet restore

## Building the Project

To build the project, use the following command:
dotnet build

To run the project, use the following command:
dotnet run
